What powers it

A production-grade ML pipeline with engineered features and a smart recommendation engine.

XGBoost Model

  • Class-imbalance weighting
  • StandardScaler + OneHotEncoder pipeline
  • L1 + L2 regularisation
  • Optimised for churn detection

Feature Engineering

  • TotalServices — active add-on count
  • ChargesPerService — cost efficiency
  • TotalCharges for lifetime value
  • Stratified train / test split

Recommendation Engine

  • Contract upgrade offers
  • Personalised billing discounts
  • Service bundle trials
  • Auto-pay & onboarding actions
